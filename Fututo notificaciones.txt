INFRAESTRUCTURA DE CORREO DISPONIBLE
--------------------------------------
- Proveedor: Microsoft 365 (Outlook con dominio corporativo @tuempresa.com)
- Servidor SMTP: smtp.office365.com / Puerto: 587 / Seguridad: STARTTLS
- Se recomienda crear una cuenta dedicada: notificaciones@tuempresa.com
  (o similar) para que el remitente sea identificable y no mezcle con correo humano

REQUISITO PREVIO (acción del admin de Microsoft 365):
  1. Crear o designar cuenta remitente (ej: notificaciones@tuempresa.com)
  2. Habilitar SMTP AUTH para esa cuenta:
     Centro de administración M365 → Usuarios → [cuenta] →
     Correo → Administrar configuración de correo → SMTP autenticado: ON
  3. Si la cuenta tiene MFA activo: generar una App Password específica
     (Seguridad → Contraseñas de aplicación) y usarla en lugar de la contraseña normal


DEPENDENCIAS A INSTALAR
------------------------
En la raíz del proyecto:

  npm install nodemailer
  npm install nodemailer-express-handlebars   # opcional, para plantillas HTML

No se requieren más dependencias. Nodemailer es compatible con Microsoft 365
via SMTP con STARTTLS sin configuración adicional.


VARIABLES DE ENTORNO A AGREGAR (.env y en Render dashboard)
-------------------------------------------------------------
  SMTP_HOST=smtp.office365.com
  SMTP_PORT=587
  SMTP_SECURE=false          # false = STARTTLS en puerto 587
  SMTP_USER=notificaciones@tuempresa.com
  SMTP_PASS=contraseña_o_app_password
  MAIL_FROM="Sistema Compras <notificaciones@tuempresa.com>"

En Render: Settings → Environment → Add Environment Variable (una por una)
NUNCA commitear estos valores al repositorio.


ARQUITECTURA RECOMENDADA
-------------------------
Crear un servicio de email centralizado en:

  backend/src/services/emailService.js

Este archivo debe:
  1. Crear el transporter de Nodemailer con la config SMTP
  2. Exportar funciones específicas para cada tipo de notificación
  3. Ser importado por las rutas que necesiten enviar correos

Ejemplo de estructura del archivo:

------------------------------------------------------------
  const nodemailer = require('nodemailer');

  const transporter = nodemailer.createTransport({
    host: process.env.SMTP_HOST,
    port: parseInt(process.env.SMTP_PORT),
    secure: false,
    auth: {
      user: process.env.SMTP_USER,
      pass: process.env.SMTP_PASS,
    },
    tls: { ciphers: 'SSLv3' }  // requerido por Office365
  });

  async function enviarOrdenCompra({ destinatario, cc, asunto, htmlBody, adjuntoPDF }) {
    return transporter.sendMail({
      from: process.env.MAIL_FROM,
      to: destinatario,
      cc: cc,
      subject: asunto,
      html: htmlBody,
      attachments: adjuntoPDF ? [{ filename: 'orden_compra.pdf', content: adjuntoPDF }] : []
    });
  }

  module.exports = { enviarOrdenCompra, /* otras funciones */ };
------------------------------------------------------------
