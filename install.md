# Instalación paso a paso (principiante)

## Opción 1: En tu computadora
1. Instala Node.js desde la web oficial.
2. Descomprime el ZIP.
3. Entra a la carpeta del proyecto.
4. Haz clic en la barra de dirección del explorador, escribe `cmd` y presiona Enter.
5. En la ventana negra ejecuta:
   npm install
6. Luego:
   npm run init-db
7. Luego:
   npm start
8. Abre tu navegador en `http://localhost:3000`

## Opción 2: Con Docker
1. Instala Docker Desktop.
2. Abre terminal en la carpeta del proyecto.
3. Ejecuta:
   docker compose up
4. Abre `http://localhost:3000`
